Основні формули математичної статистики

для дискретної випадкової величини

Дискретна випадкова величина-це випадкова величина, множина значень якої скінченна.

  1. Розмах вибірки:

    $$R=x_{max}-x_{min},$$ $$де\;x_{min}\;-\;значення\;найменшої\;варіанти,\;x_{max}\;-\;значення\;найбільшої\;варіанти.$$

  2. Відносна частота варіанти:

    $$w_{i}=\frac{n_{i}}{n},$$ $$де\;n_{i}\;-\;частота\;варіанти,\;n\;-\;сума\;всіх\;частот.$$

  3. Вибіркове середне:

    $$\overline{x}=\frac{1}{n}\cdot \sum_{i=1}^{n}x_{i}$$

  4. Математичне сподівання:

    $$M(X)=\sum_{i=1}^{n}x_{i}\cdot w_{i}$$

  5. Дисперсія:

    $$D=\sum_{i=1}^{n}x_{i}^{2}\cdot w_{i}-\overline{x}^2$$

  6. Виправлена дисперсія:

    $$S^{2}=\frac{1}{n-1}D$$

  7. Середне квадратичне відхилення:

    $$\sigma=\sqrt{D}$$

  8. Виправлене середне квадратичне відхилення:

    $$S=\sqrt{S^{2}}$$

  9. Коефіціент варіації:

    $$V=\frac{S}{\overline{x}}\cdot 100\%$$

  10. Початкові моменти:

    $$V_{1}=\sum_{i=1}^{n}x_{i}\cdot w_{i},$$ $$V_{2}=\sum_{i=1}^{n}x_{i}^{2}\cdot w_{i},$$ $$\cdots$$ $$V_{k}=\sum_{i=1}^{n}x_{i}^{k}\cdot w_{i}$$

  11. Центральні моменти:

    $$\mu _{1}=0,$$ $$\mu _{2}=V_{2}-V_{1}^{2},$$ $$\mu _{3}=V_{3}-3\cdot V_{1}\cdot V_{2}+2\cdot V_{1}^{3},$$ $$\mu _{4}=V_{4}-4\cdot V_{1}\cdot V_{3}+6\cdot V_{1}^{2}\cdot V_{2}-3\cdot V_{1}^{4}$$

  12. Асиметрія:

    $$A_{s}=\frac{\mu _{3}}{\sigma ^{3}}$$

  13. Ексцес:

    $$E_{k}=\frac{\mu _{4}}{\sigma ^{4}}-3$$

  14. Довірчій інтервал для математичного сподівання:

    $$M(X)-\frac{t_{\gamma }\cdot S}{\sqrt{n}}< M(X)< M(X)+\frac{t_{\gamma }\cdot S}{\sqrt{n}},$$ $$де\;t_{\gamma }(\gamma ,n)\;-\;табличне\;значення\;функції\;Стьюдента,\;\gamma \;-\;довірча\;ймовірність.$$

  15. Довірчій інтервал для середнього квадратичного відхилення:

    $$S\cdot (1-q)<\sigma < S\cdot (1+q),\;(якщо\;q< 1),$$ $$0<\sigma < S\cdot (1+q),\;(якщо\;q> 1),$$ $$де\;q(\gamma ,n)\;-\;табличне\;значення.$$
